# Secondary Product Image Usage

Last updated: 2026-06-04

This file records the one-image-one-use split for the secondary product categories. Do not collapse these back into shared matrix crops.

## Hard Rule

- Each SKU, menu panel, and product-page hero must use its own image file.
- Do not reuse a category hero as a menu image.
- Do not reuse a menu image as a product-page hero.
- Do not crop multiple SKU images out of one broad scene.
- SKU product files should be independent transparent PNGs generated from flat chroma-key sources.
